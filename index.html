Page content

The main page content appears inside the body tag. HTML contains several elements that allow you to properly structure and format your content, which we'll cover later.

The content model

Metadata: Setting up presentation or behviors. It's in the head. Meta tags. Script. Title, etc.

Embedded: Imports other resources into the documment. Canvas, iframe, etc

Interactive: Focuses on any sort of user interaction. Such as anchor or link tags, buttons, etc

Heading: Headders of the sections

Phrasing: Text of the document and any elements used to mark up document level structures

Flow: Elements included in normal flow of the document. It's basically includes everything except some metadata.

Sectioning: Defines the scope such as headings and footers.
